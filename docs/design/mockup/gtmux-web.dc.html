gtmux · web 浏览器镜像

任意电脑上的只读雷达

「Open on computer」生成一个一次性、view-only 链接,在任意电脑的浏览器里看雷达 + 读 pane——单二进制内嵌、xterm.js 渲染、颜色取自你的 Ghostty 配置。与手机/菜单栏同一套状态语言,但为桌面宽屏 + 真键盘而设计;不假装能输入,避免权限与误操作。实现见 internal/server/web/。

🔒 view-only · 输入请用手机或回到 Mac
定位
只读镜像,不是完整客户端。价值:大屏 + 真键盘盯全局、细读 pane;一次性链接、免装 app。输入仍走手机/Mac —— web 不假装能写。
桌面适配
· 内容居中到 820px 列,两侧留白。
· xterm.js 真彩 + CJK 宽度;字体/字号可选(内嵌 5 款 mono)。
· 颜色取自 Ghostty 配置,网页与终端同色系。
· 复制:终端原生选择;链接顶部 view-only 常驻。
键盘 · 安全
· j/k 选行、Enter 进 pane、Esc 返回、/ 搜索。
· waiting 时浏览器标题闪烁 + favicon 红点(可选)。
· 链接过期/撤销即 410;空/断连用与 app 一致的 gate 文案。
