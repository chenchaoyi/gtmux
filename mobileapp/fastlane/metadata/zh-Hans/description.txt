gtmux 是你 Mac 上 CLI agent 的雷达与指挥中心——既包括 Claude、Codex 这样的编码 agent，也包括任何跑在终端里的 agent。在 Mac 上运行 gtmux serve，手机配对一次，就能把整支舰队随身带走。

看见（SEE）
把每个 tmux 窗格和 CLI agent 汇成一张实时雷达，用颜色区分状态——等你输入、正在工作、空闲。一眼就知道谁在等你。

操控（STEER）
从手机直接回话，或把命令发到终端。把某个 agent 拉到眼前。不用走回 Mac 前就能解堵。

中控（SUPERVISE）
一个帮你盯全局的中控：它消化每个 agent 在做什么，只把真正需要你的浮上来，还能替你分诊、派活——让繁忙的舰队在不重要时保持安静。

此外
• agent 需要你或跑完的那一刻才推送，不提前打扰。
• 灵动岛——把当前 agent 的状态留在锁屏和灵动岛上。
• 安全分享——一个受限、可撤销的链接，让同事只看到你允许的会话、只在你允许的窗格里输入。输入默认关闭。
• 支持英文与简体中文。

工作原理
gtmux 连接你自己 Mac 上的一个小服务（gtmux serve），走你自己的网络、VPN 或隧道——就像连接你自己机器的 SSH／终端客户端。不上传任何数据到我们、没有账号、手机上不运行任何代码。访问由你掌握、可随时撤销的令牌把关。

隐私
gtmux 不收集任何数据。你的会话与令牌只保存在你的设备和你的 Mac 上。
