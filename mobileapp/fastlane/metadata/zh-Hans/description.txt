gtmux 是你 Mac 上 tmux 会话与编码 agent 的指挥中心。在 Mac 上运行 gtmux serve，手机配对一次，就能一眼看到每个会话的状态——哪个 agent 在等你、哪个在干活、哪个空闲。

gtmux 能做什么
• 实时雷达——把每个 tmux 窗格和编码 agent（Claude、Codex 等）汇总成一个列表，用颜色区分状态：等你输入、正在工作、空闲。
• 该提醒才提醒——agent 需要你输入或跑完时才推送，不打扰。
• 直接回话——从手机把回复或命令发到终端，不用走到 Mac 前就能给 agent 解堵。
• 灵动岛——把当前 agent 的状态留在锁屏和灵动岛上。
• 安全分享——把一个受限、可撤销的链接交给同事：他们只能看到你允许的会话、只能在你允许的窗格里输入。输入默认关闭。
• 支持英文与简体中文。

工作原理
gtmux 连接你自己 Mac 上的一个小服务（gtmux serve），走你自己的网络、VPN 或隧道。它就像连接你自己机器的 SSH／终端客户端——不上传任何数据到我们、没有账号、手机上不运行任何代码。访问由你掌握、可随时撤销的令牌把关。

隐私
gtmux 不收集任何数据。你的会话与令牌只保存在你的设备和你的 Mac 上。
